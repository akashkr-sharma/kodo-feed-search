# Install
> Pre-requirement
**node.js**, **yarn**

```bash
$ git clone https://github.com/akashkr-sharma/koto-feed-search
$ cd koto-feed-search
$ yarn install
```

## Configure and Run

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
